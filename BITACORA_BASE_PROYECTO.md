# BITÁCORA BASE DEL PROYECTO

## Datos generales
- Proyecto: Demo Beta Landing
- Base técnica: Next.js 14 + React + TypeScript
- Estilo visual: limpio/profesional, tokens de diseño, Poppins
- Fecha de inicio: 2026-04-12
- Estado actual: planificación y definición de alcance

## Propósito de esta bitácora
Documentar decisiones, avances, bloqueos y cambios para convertir la demo en un proyecto sólido y escalable.

## Objetivo principal
Presentar una demo convincente al dueño que refleje claramente su idea, valide la dirección visual y sirva como base para fases posteriores.

## Alcance confirmado (beta)
- Landing one-page enfocada en conversión.
- Arquitectura moderna y limpia.
- UI consistente basada en tokens.
- Responsive completo.

## No incluido por ahora
- Multi-tenant.
- Login/panel (auth).
- Prisma/MySQL en etapa inicial.

## Decisiones técnicas tomadas
1. Usar App Router de Next.js 14 para escalabilidad.
2. Mantener TypeScript desde el inicio para reducir deuda técnica.
3. Priorizar CSS propio con tokens sobre librerías pesadas.
4. Diseñar beta sin complejidad de backend persistente.

## Decisiones de diseño tomadas
1. Tipografía principal: Poppins.
2. Paleta: neutros + indigo para foco/acciones.
3. Jerarquía por tarjetas y bloques cortos.
4. Lenguaje visual profesional, claro y sin sobrecarga.
